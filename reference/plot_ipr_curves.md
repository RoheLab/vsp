# Plot IPR curves.

Plot IPR curves.

## Usage

``` r
plot_ipr_curves(localization, indices = NULL)
```

## Arguments

- localization:

  A `localization_stats` object.

- indices:

  A vector of integers specifying which singular vectors to plot.

## Value

A `ggplot2` object.
